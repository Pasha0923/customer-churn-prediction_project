# Прогнозування відтоку клієнтів для телекомунікаційної компанії (customer-churn-prediction_project )

## 📌 Опис проекту
Проект спрямований на прогнозування відтоку клієнтів телекомунікаційної компанії.  
Модель передбачає ймовірність того, що клієнт або залишить компанію найближчим часом або залишиться з компанією  і надає рекомендації щодо утримання клієнтів на основі їх характеристик.
## 🎯 Мета проекту 
Розробити модель машинного навчання для прогнозування ймовірності відтоку клієнтів телекомунікаційної компанії на основі історичних даних про їхнє використання послуг та демографічних характеристик.

## Проект включає:
- Аналіз даних та візуалізацію ознак (EDA)
- Попередня обробка даних (preprocessing)
- Розробка та порівняння моделей машинного навчання (LogisticRegression, RandomForest, GradientBoosting, XGBoost, LightGBM, CatBoost)
- Оптимізацію гіперпараметрів через GridSearch 
- Вибір найкращої моделі
- Інтерфейс для ручного прогнозування та інтерактивної аналітики через Streamlit

## 🛠 Етапи реалізації
1. **EDA**:
   - Перевірка на пропуски, дублікати 
   - Аналіз даних на викиди
   - Аналіз впливу ознак на відтік   
   - Розподіл ознак , кореляція між ознаками
2. **PREPROCESSING**
   - Обробка пропусків 
   - Логарифмування ознак із скошеним розподілом
   - Підготовка даних для подачі в модель
   - Масштабування необхідних числових ознак
3. **MODELLING**
   - Поділ на train/test зі стратифікацією за цільовою змінною
   - Крос-валідація , навчання та порівняння моделей Логістична регресія, Random Forest, градієнтний бустинг, XGBoost, LightGBM, CatBoost 
   - Обчислення метрик моделей : accuracy , precision , recall , F1-score, ROC-AUC
4. **Тюнінг моделей**
   - GridSearch для XGBoost та LightGBM
5. **Вибір кращої моделі**
   - LightGBM
6. **Інтерфейс Streamlit**
   - Ручне прогнозування з персоналізованими рекомендаціями щодо утримання
   - інтерактивна аналітика (Feature Importance, кореляція між метриками , таблиця результатів метрик моделей )
   - Візуалізація результатів прогнозування

## ⚡ Інструкція із запуску

1. **Склонуйте репозиторій**
```bash
git clone https://github.com/Pasha0923/customer-churn-prediction_project.git
cd customer-churn-prediction_project
```
2. **Встановления залежностей (poetry or pip):**

using poetry:
```bash
poetry install
```
using pip:
```bash
pip install -r requirements.txt
```
3. **Запуск Streamlit інтерфейса**
```bash
streamlit run app.py
```
У браузері після запуску перейдіть по локальному URL http://localhost:8501/

4. **Навігація по інтерфейсу**

Аналітика даних – вкладка Аналітика

Ручне прогнозування – вкладка Прогноз
